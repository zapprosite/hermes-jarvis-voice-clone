# Hermes Jarvis Voice Clone

**Voice model jarvis-clone-trimmed (ONNX + config) para hermes-agent**

## Install

```bash
pip install hermes-jarvis-voice-clone
hermes_jarvis_voice_clone_bootstrap  # se disponível
```

## Compatibilidade

- Python 3.11+

- Ubuntu 22.04+
